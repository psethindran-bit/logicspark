LEVEL 1 — BASIC

BEGINNER
HTML & CSS Foundations
Master the structure of web pages with semantic HTML5 and modern CSS styling techniques for professional layouts.
  • Create semantic headings (h1-h6), paragraphs, and lists
  • Build navigation menus with <nav> and links
  • Embed images, videos, and responsive media
  • Style with CSS colors, Google Fonts, and Flexbox basics
  • Understand box model, padding, margins, and positioning
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <header>
        <h1>Welcome to Logic Spark</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#home">Home</a></li>
            <li><a href="#about">About</a></li>
        </ul>
    </nav>
</body>
</html>
Pro Insight: Always use semantic HTML for better SEO and accessibility. Validate your code at validator.w3.org.[web:6]
BEGINNER
CSS Flexbox & Grid
Create responsive layouts using modern CSS Flexbox and Grid systems for pixel-perfect designs across all devices.
  • Master Flexbox properties: justify-content, align-items
  • Create CSS Grid layouts with grid-template-columns
  • Build responsive navigation and card layouts
  • Use media queries for mobile-first design
  • Combine Flexbox (1D) and Grid (2D) layouts effectively
.container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
    padding: 20px;
}

.card {
    display: flex;
    flex-direction: column;
    background: #111;
    padding: 20px;
    border-radius: 10px;
}
Pro Insight: Flexbox excels for 1D layouts (rows/columns), Grid for 2D layouts. Test on real devices.[web:6]

LEVEL 2 — MEDIUM

INTERMEDIATE
JavaScript Basics
Add interactivity with modern JavaScript ES6+ syntax, functions, and event handling for dynamic web experiences.
  • Declare variables with let/const, understand scope
  • Work with arrays, objects, and modern methods (map, filter)
  • Create arrow functions and template literals
  • Handle events: click, input, submit
  • Debug with console.log(), breakpoints
// Modern ES6+ JavaScript
const users = ['Alice', 'Bob', 'Charlie'];

const uppercased = users.map(name => name.toUpperCase());
console.log(uppercased); // ['ALICE', 'BOB', 'CHARLIE']

// Event handler
document.querySelector('button').addEventListener('click', () => {
    alert('Button clicked!');
});
Pro Insight: Use strict mode ('use strict') and modern ES6+ syntax. Practice on CodePen or JSFiddle.[web:9]
INTERMEDIATE
DOM Manipulation
Dynamically update page content, styles, and structure using JavaScript DOM APIs for interactive UIs.
  • Select elements: querySelector, getElementById
  • Modify textContent, innerHTML, attributes
  • Style manipulation: element.style.property
  • Create/remove elements dynamically
  • Event delegation for performance
// Live DOM manipulation example
const button = document.querySelector('#myButton');
const output = document.querySelector('#result');

button.addEventListener('click', () => {
    const newDiv = document.createElement('div');
    newDiv.textContent = 'New element added!';
    newDiv.style.color = 'cyan';
    newDiv.style.padding = '10px';
    
    document.body.appendChild(newDiv);
    
    output.innerHTML = `Total elements: ${document.querySelectorAll('div').length}`;
});
Pro Insight: Use textContent over innerHTML to prevent XSS. Event delegation improves performance.[web:9]

ADVANCED
APIs & AJAX
Fetch real-time data from REST APIs and display dynamic content without page reloads using modern Fetch API.
  • Use modern fetch() API with async/await
  • Parse JSON responses and handle errors
  • Update DOM with API data dynamically
  • Implement loading states and error handling
  • CORS handling and API authentication
// Fetch API with async/await
async function fetchUsers() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/users');
        const users = await response.json();
        
        users.slice(0, 5).forEach(user => {
            const div = document.createElement('div');
            div.innerHTML = `<h3>${user.name}</h3><p>${user.email}</p>`;
            document.body.appendChild(div);
        });
    } catch (error) {
        console.error('API Error:', error);
    }
}

fetchUsers();
Pro Insight: Always handle errors and show loading states. Use JSONPlaceholder for practice APIs.[web:10]
